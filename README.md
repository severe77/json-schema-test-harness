# JSON Schema Test Harness (AJV)

This repository contains a small test harness for running tests from the official JSON Schema Test Suite using the AJV validator in Node.js.

The script reads test files from the suite, runs validation using AJV, and prints PASS / FAIL results along with a final summary.

## Purpose

This project was created as part of the qualification task for the **“Unify the Test Suite”** project idea in the JSON Schema community (GSoC).

The goal was to build a simple tool that can:

* load tests from the JSON Schema Test Suite
* compile schemas using AJV
* validate test data against those schemas
* compare actual results with expected results
* print PASS / FAIL output and a summary

## Project Structure

Main files in this repository:

* **runner.js** – script that runs the tests
* **package.json** – project dependencies and npm configuration
* **package-lock.json** – dependency lock file generated by npm
* **JSON-Schema-Test-Suite/** – cloned copy of the official JSON Schema Test Suite

The harness reads test files directly from the `tests` directory inside the suite.

## Requirements

* Node.js (v18+ recommended)
* npm

## Installation

Clone the repository and install dependencies:

```
npm install
```

## Running the Test Harness

Run all tests:

```
node runner.js
```

You can also run tests for a specific draft:

```
node runner.js draft7
```

Example:

```
node runner.js draft2020-12
```

## Example Output

```
Running draft7/maximum.json
PASS draft7/maximum.json :: valid maximum
PASS draft7/maximum.json :: invalid maximum

Summary
Total: 6448
Passed: 2612
Failed: 3836
```

## Notes

Some tests may fail depending on the level of JSON Schema draft support in the validator.

AJV mainly focuses on newer drafts of the JSON Schema specification, so behavior may differ when running tests for older drafts.

These failures do not indicate a problem with the harness itself.
